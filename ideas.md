**Design Movement**: 자연주의 미니멀리즘 (Nature-Informed Minimalism)

**Core Principles**:
1. 자연에서 영감받은 유기적 형태와 부드러운 곡선
2. 데이터 신뢰성을 강조하는 깔끔한 정보 계층
3. 반려동물의 따뜻함과 플랫폼의 과학적 정확성의 균형
4. 여백을 적극 활용한 호흡감 있는 레이아웃

**Color Philosophy**:
- Primary: Forest Green (#2D7D46) - 자연, 신뢰, 건강
- Secondary: Sage (#7FB069) - 부드러운 보조 포인트
- Background: Warm Cream (#FAF7F2) - 따뜻하고 친근한 배경
- Surface: White (#FFFFFF) - 카드 및 컨텐츠 영역
- Text: Charcoal (#2C2C2C) - 가독성 높은 본문

**Layout Paradigm**:
- 비대칭 히어로: 텍스트 좌측, 이미지 우측 오버플로우
- 카드 기반 서비스 소개 (3열 그리드)
- 스텝 프로세스는 수평 타임라인
- 리포트 섹션은 데이터 대시보드 스타일

**Signature Elements**:
1. 발바닥 아이콘 모티프 (로고 및 구분선에 활용)
2. 부드러운 초록 그라디언트 배경 섹션
3. 데이터 차트/레이더 차트 시각화 요소

**Interaction Philosophy**:
- 스크롤 기반 애니메이션으로 데이터가 살아나는 느낌
- 호버 시 카드가 살짝 떠오르는 효과
- CTA 버튼은 초록 배경 + 흰 텍스트로 명확한 행동 유도

**Animation**:
- 페이지 진입: fade-in + slide-up (0.3s ease-out)
- 카드 호버: translateY(-4px) + shadow 강화
- 숫자 카운터: 스크롤 진입 시 0에서 목표값으로 증가
- 레이더 차트: 점진적 그리기 애니메이션

**Typography System**:
- 헤드라인: Noto Sans KR Bold (700) - 한국어 가독성 최우선
- 서브헤드: Noto Sans KR SemiBold (600)
- 본문: Noto Sans KR Regular (400)
- 강조 숫자: 큰 사이즈 + Bold로 데이터 시각화
